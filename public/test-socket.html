Socket.IO Bağlantı Testi

📋 Kullanım Talimatları:

  1. ÖNEMLİ: Bu sayfa HTTP üzerinden açılmalıdır: http://localhost:3000/test-socket
  2. Swagger'da (/api-docs) kullanıcı girişi yapın
  3. JWT token'ı kopyalayın ve yukarıdaki alana yapıştırın
  4. "Bağlan" butonuna tıklayın
  5. "Test Login" butonuna tıklayarak kullanıcıyı online yapın
  6. "Durum Kontrol Et" ile online durumunu kontrol edin
  7. YENİ: Socket.IO event'lerini test edin

🔧 Socket.IO Event Testleri:

  • Konuşmaları Getir: Mevcut konuşmaları listeler ve ID'leri gösterir
  • Odaya Katıl: Konuşma odasına katılma (Konuşma ID gerekli)
  • Mesaj Gönder: Gerçek zamanlı mesaj gönderme (Alıcı ID gerekli)
  • Yazıyor: Typing indicator testi (Alıcı ID gerekli)
  • Okundu İşaretle: Mesaj okundu bildirimi (Mesaj ID ve Gönderici ID gerekli)

⚠️ Dosyayı doğrudan tarayıcıda açmayın! HTTP üzerinden erişin.

Bağlantı durumu: Bağlı değil
